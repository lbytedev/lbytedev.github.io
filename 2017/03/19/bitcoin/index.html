第四周-BitCoin

发表于 2017-03-19   |   分类于 Gossip   |  

区块链是一种社会思潮，它预示着人类社会转型、换代的新时代的到来。

入职ViaBTC第一周，在阅读区块链，技术驱动金融 这本书，对比特币有了一个基本的认识。之前从来没有想过，为什么会有经济危机？国家为什么要超发货币？跨国交易为什么那么麻烦？更没有想过可以有一种全新的货币系统可以去解决现有货币系统存在的一些问题。当然这种货币不一定是比特币，比特币本身也存在很多问题，例如比特币每秒只支持3笔交易，无法支持大多数人都使用比特币进行交易。最近比特币的分叉事件也导致了比特币价格的下跌。为什么会分叉？是因为bitcoin core和比特币社区产生了一些分歧，bitcore core的存在就是不合理的，比特币本应该是一个去中心化的系统，有了bitcoin core的存在就谈不上去中心化了吧，比特币的发展方向都很大程度掌握在这群人手上。无论比特币存在一些怎样的问题，甚至比特币价格归于0了，我都相信，区块链技术会将这个世界带入一个新的时代。

第一周，和同事们有了简单的接触，大家能力都很强，独当一面，有一定的压力，自己也要做得更好，才不至于给大家拖后腿。团队的氛围也不错，由于我们处在一个高速发展又动荡的行业，常常都有值得探讨的事情发生，例如最近比特币分叉的事件就引起了同事们对中本聪本意的探讨，比特币是否存在不可解决的问题，其它虚拟货币例如ETH是否会有更好的发展前景。

走进一个新的领域，很兴奋。

感觉有收获？
