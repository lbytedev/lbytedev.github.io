第五周-做事的方式和状态

发表于 2017-03-27   |   分类于 Gossip   |  

花更多的时间并不是努力的方式，关注做事的方式和状态

这周开始搭建新产品的架构，还算顺畅。产品的难点在于一个k线图，觉得自己有时候会走入一个思维误区，总希望拿到一个现成的东西改吧改吧就能用，这样会导致最终的出来的东西灵活性差，以及虽然将东西做出来了，但是对原理并不了解，没有取得成长。

公司为了新产品的上线，每周都是996，这样还挺累人的，效率低下，我并不排斥996，其实来了一个创业公司，我也希望自己能有创业者的心态，多花些时间，将产品尽可能的做到完美。总之，来了创业公司，就要培养自己作为一个创业者的能力。只是要做好事情，增加时间是不够的，还得提升效率，下周开始要抽时间运动起来。

下周重点，调整做事的方式和状态。
