Github更改邮箱引起的权限问题

发表于 2016-03-29   |   分类于 踩过的坑   |  

某天准备使用hexo推送文章到blog的时候发现推送失败，百思不得其解，前一篇文章是成功推送过的，ssh-key也早就配置好了，为嘛突然会有问题呢？错误提示:

  • Permission denied(publickey).fetal:Could not read from remote reposity.Please make sure you have the correct access rights and the repository exists.

后来发现是我手痒改了Github的邮箱,用新邮箱重新生成一个ssh-key到Github就好了。如何生成请参考：Github官方说明
